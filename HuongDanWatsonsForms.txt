## 📊 TỔNG QUAN DỰ ÁN

### Thông tin cơ bản
- **Công nghệ:** .NET 8.0 WinForms + SQL Server
- **Kiến trúc:** Clean Architecture (Service/Repository Pattern)
- **ORM:** Dapper
- **Tổng số dòng code C#:** 19,477 dòng
- **Tổng số dòng code SQL:** 1,516 dòng
- **Tổng số files C#:** 86 files
- **Thời gian phát triển:** 3 tháng (9/2025 - 12/2025)

### Phạm vi chức năng
Hệ thống quản lý toàn diện cho cửa hàng bán lẻ Watsons, bao gồm:
- Quản lý bán hàng (POS)
- Quản lý kho hàng (nhập/xuất/tồn)
- Kế toán tài chính (thu/chi/bút toán)
- Báo cáo quản trị (doanh thu, lợi nhuận, công nợ)
- Quản lý danh mục (sản phẩm, khách hàng, nhà cung cấp)
- Phân quyền người dùng

### 2. Database Architecture

**Cơ sở dữ liệu:** QL_Watsons

**Thống kê:**
- **18 Tables** (bảng dữ liệu chính)
- **4 Views** (báo cáo tổng hợp)
- **7 Stored Procedures** (nghiệp vụ phức tạp)

**Cấu trúc bảng chính:**

#### A. Hệ thống tài khoản kế toán (3 cấp)
```
TaiKhoanCap1 (8 tài khoản)
├── TaiKhoanCap2 (11 tài khoản)
    └── TaiKhoanChiTiet (58 tài khoản chi tiết)
```

**Ví dụ cấu trúc:**
- Cấp 1: `1 - Tài sản ngắn hạn`
  - Cấp 2: `13 - Phải thu khách hàng`
    - Chi tiết: `131.KH001, 131.KH002, ...` (20 khách hàng)

#### B. Master Data
```
DonViTinh           # Đơn vị tính (Hộp, Chai, Gói...)
LoaiSanPham         # Loại SP (Mỹ phẩm, Thực phẩm...)
HangSanPham         # Thương hiệu (Nivea, Olay...)
SanPham             # 120 sản phẩm
NhaCungCap          # 15 nhà cung cấp
KhachHang           # 20 khách hàng
NhanVien            # 15 nhân viên
```

#### C. Chứng từ & Bút toán
```
LoaiChungTu         # PNK, PBH, PT, PC
ChungTu             # Chứng từ header
ButToan             # Chi tiết bút toán (TKNo, TKCo, SoTien)
```

#### D. Khác
```
SoDuDauKy          # Số dư đầu kỳ tài khoản
KhuyenMai          # Chương trình khuyến mãi
CaLamViec          # Ca làm việc thu ngân
TaiKhoan           # User accounts
```

- **Client-side:** TextBox validate, Required fields

- **Database:** FK constraints, CHECK constraints

- **User-friendly messages:** MessageBox.Show

## 🐛 BUGS FIXED

### Bug #1: usp_TaoPhieuThuNo logic sai
**Vấn đề:** Thu tiền từ KH làm tăng nợ thay vì giảm
```sql
-- SAI:
UPDATE SoDuDauKy SET DuCoDau += @SoTien WHERE MaTK = @MaTK_KH

-- ĐÚNG:
UPDATE SoDuDauKy SET DuNoDau -= @SoTien WHERE MaTK = @MaTK_KH
```
**Giải thích:** TK 131 có tính chất Nợ, thu tiền làm giảm Nợ

### Bug #2: Thiếu TK 334 (Phải trả NLĐ)
**Vấn đề:** TK 334.NVxxx được tạo nhưng không có TK cha 334
```sql
-- Đã thêm:
INSERT INTO TaiKhoanCap2 VALUES ('334', N'Phải trả người lao động', '3', 'Co')
```

### Bug #3: FK constraint timing
**Vấn đề:** SanPham.MaTK FK tham chiếu TaiKhoanChiTiet trước khi TK được tạo
**Fix:** Di chuyển FK sang ALTER TABLE sau khi tất cả bảng tạo xong

### Bug #4: Invalid data references
**Vấn đề:** DATA_INSERT.sql gọi SP với NCC/KH không tồn tại
**Fix:** Xóa tất cả EXEC invalid (NCC016-020, KH021-030)

### Bug #5: Drop tables with FK constraints
**Vấn đề:** DTB.sql không thể drop tables có FK
**Fix:** Thêm `sp_MSforeachtable 'ALTER TABLE ? NOCHECK CONSTRAINT ALL'`
